# 📄 Use Case Description
## Descriptions
| Use Case | Description|
|---|---|
| 회원 가입 | 비회원은 서비스 이용을 위해 회원가입을 해야 한다. 회원가입시, ID, 비밀번호, 전화번호, 결제 수단, 선호 자전거 유형(일반/전기)을 필수로 입력해야 한다. 시스템은 입력된 결제 수단 정보를 외부 결제 시스템에 전달하고 토큰을 발급받는다. |
| 회원 탈퇴 | 회원은 언제든지 서비스를 탈퇴할 수 있다. |
| 로그인 | 사용자는 ID와 비밀번호로 로그인할 수 있다. 이때, 관리자는 주어진 ID(admin)와 비밀번호(admin)으로 로그인이 가능하다. |
| 로그아웃 | 로그인한 사용자는 언제든지 서비스 로그아웃을 할 수 있다. 로그아웃 시 시스템 접속이 종료된다.|
| 대여소 등록 | 관리자는 대여소 정보를 등록할 수 있다. 등록 시 입력 정보는 대여소 이름, 위치(도시 및 상세주소), 자전거 보관 가능 수량, 운영 시간이다. 이때 도시는 시/군/구로 구분하며, 이미 등록된 대여소와 동일한 이름은 사용할 수 없다. |
| 대여소 리스트 조회 | 관리자는 등록된 대여소의 리스트를 볼 수 있다. 대여소 리스트 중, 특정 대여소 항목의 삭제 버튼을 눌러 등록한 대여소를 삭제할 수 있다. |
| 관리자용 대여소 상세정보 조회 | 관리자는 대여소 리스트 조회 화면에서 특정 대여소 항목을 선택하여, 해당 대여소의 상세정보(대여소 이름, 위치(도시 및 상세주소), 자전거 보관 가능 수량, 운영 시간)를 확인할 수 있다. |
| 자전거 등록 | 관리자는 새로운 자전거 정보를 등록할 수 있다. 등록 시, 자전거 ID, 자전거 제품명, 유형(일반/전기), 소속 대여소, 상태(사용 가능/수리 중)을 입력해야 한다. |
| 자전거 리스트 조회 | 관리자는 등록된 자전거의 리스트를 볼 수 있다. 자전거 리스트의 각 항목은 자전거 ID, 자전거 제품명, 유형(일반/전기), 소속 대여소, 상태(사용 가능/수리 중)를 포함한다. 이 때, 리스트는 자전거 ID 순으로 정렬되어 출력된다. 관리자는 리스트 중 원하는 자전거 항목의 삭제 버튼을 눌러 등록된 자전거를 삭제할 수 있다. |
| 자전거 상세정보 조회 | 관리자는 등록된 자전거 리스트에서 특정 자전거를 선택해서 상세정보를 조회할 수 있다. 상세정보에는 등록 시 입력했던 정보인 자전거 ID, 자전거 제품명, 유형(일반/전기), 소속 대여소, 상태(사용 가능/수리 중)가 포함된다. |
| 대여소 검색 | 회원은 특정 이름의 대여소를 검색할 수 있다. 검색어로 입력된 대여소 이름과 정확히 일치하는 대여소 하나가 검색 결과로 출력된다. |
| 회원용 대여소 상세정보 조회 | 회원은 검색된 대여소 리스트에서 특정 대여소를 선택하여 상세정보를 조회할 수 있다. 상세정보 화면에는 대여소의 이름, 위치(도시 및 상세주소), 자전거 보관 가능 수량, 운영 시간, 사용 가능한 자전거 목록이 출력된다. 자전거 목록에는 수리 중인 자전거를 제외한 모든 자전거가 포함되며, 각 항목에는 자전거 ID, 제품명, 유형(일반/전기), 소속 대여소, 상태(사용 가능/수리 중)가 표시된다. 목록은 자전거 ID 순으로 정렬된다. <br> 대여 가능한 자전거가 있을 경우, 회원은 특정 자전거를 선택하여 즉시 대여할 수 있다. 대여 중인 자전거의 경우, 예약 대기를 신청할 수 있다. 대여 또는 예약대기 신청이 완료되면, 대여소와 자전거 정보를 포함한 문자 전송을 문자 발송 시스템에 요청한다. |
| 자전거 대여 정보 조회 | 회원은 현재 자신이 대여 중인 자전거 정보를 리스트로 조회할 수 있다. 리스트의 각 항목은 대여소 이름, 위치, 자전거 ID, 제품명, 자전거 유형이며, 정렬 기준은 자전거 ID 이다. <br> 회원은 특정 자전거를 반납할 수 있다. 반납 시, 결제 시스템에 해당 이용 요금 결제를 요청한다. <br> 반납된 자전거에 대기 예약한 회원이 있는 경우, 이메일 시스템에 이메일 발송을 요청하여 대기 1순위 회원에게 알린다. <br> 반납 후, 회원은 위치 기반 식당 예약 시스템에 리디렉션 되어 식당 추천을 받고 예약을 할 수 있다. |
| 자전거 예약 대기 정보 조회 | 회원은 자신이 예약 대기 중인 자전거 리스트를 조회할 수 있다. 각 항목은 대여소 이름, 대여소 위치, 자전거 ID, 자전거 제품명, 자전거 유형이다. 또한, 각 예약대기 항목에 대해 개별적으로 취소할 수 있다. |
| 결제 요금 조회 | 회원은 마지막으로 반납된 건에 대한 대여 시간 및 요금을 조회할 수 있다. |
| 회원용 자전거 대여 기록 조회 | 회원은 본인의 과거 자전거 대여 기록 리스트를 조회할 수 있다. 대여 기록 리스트는 기본적으로 반납 날짜별로 묶여서 최근순으로 정렬된다. 회원이 대여소별 정렬 버튼을 누르면 대여 기록 리스트가 대여소별로 묶여서 대여소 이름 오름차순으로 정렬된다. 같은 대여소의 대여 기록들은 최근순으로 정렬된다. 이후 다시 반납 날짜별로 대여 기록 리스트를 정렬할 수 있다. 대여 기록의 각 항목은 대여소 이름, 대여소 위치(도시 및 상세주소), 자전거 ID, 자전거 제품명, 자전거 유형(일반/전기), 자전거 반납 날짜를 포함한다. 회원은 원하는 대여 기록 항목의 삭제 버튼을 눌러 대여 기록을 삭제할 수 있다. |
| 관리자용 자전거 대여 기록 조회 | 관리자는 자전거 대여 기록 리스트를 조회할 수 있다. 기본적으로 반납 시간 기준 최근순으로 대여 기록 리스트가 정렬된다. 대여 기록의 각 항목은 대여소 이름, 대여소 위치(도시 및 상세주소), 자전거 ID, 자전거 제품명, 자전거 유형(일반, 전기)을 포함한다. 관리자가 지역별 정렬 버튼을 누르면 대여 기록들은 대여소 위치를 기준으로 도시별로 묶여서 정렬된다. 이 때, 도시 그룹은 오름차순으로 정렬되고, 같은 도시인 대여 기록들은 주소의 오름차순으로 정렬된다. 이후 다시 최근순으로 대여 기록 리스트를 정렬할 수 있다. |
| 자전거 대여 금액 및 횟수 조회 | 관리자는 ‘최근 1주일’, ‘최근 1달’, ‘최근 1년’ 기간 단위를 표시하는 리스트 중 하나의 항목을 선택하여 해당 기간 동안 발생한 총 대여 금액 및 대여 횟수를 조회할 수 있다. 이후 다른 기간 단위를 선택하여 해당 항목에 대해서도 마찬가지로 조회할 수 있다.
